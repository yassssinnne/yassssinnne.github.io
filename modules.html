Khan Academy Intro to JS: Drawing & Animation

This module allows students to learn the fundamentals of the JavaScript language and introduces the ProcessingJS library to allow students to create drawings and animations. JavaScript is a widely used language and can be incorporated with HTML to make webpages or is also seen being used for libraries such as p5.js. Therefore, building a solid foundation in this language was necessary and possible with this module. The most significant impact of this module for me, was learning about variables, functions, logic & if statements, looping, arrays, objects and object-oriented design as I have continued to use these principles on a daily basis when coding. These principles allow for me to create more complex data for my programs as seen later in some of my Advanced JS projects.

Khan Academy HTML/JS: Making webpages interactive

This module combines knowledge from JS, HTML & CSS and allows students to be able to make interactive webpages with the help of the JavaScript DOM API library. The students are taught how to use DOM API to access elements on the webpage, make modifications, make the webpage react to user events and to animate parts of the webpage. Although using DOM API is an excellent introduction to making interactive webpages, moving forward students may want to continue to learn by learning how to use jQuery instead as working with it is a little faster. The most significant part of this module for me, was the introduction to using JavaScript libraries in webpages as it allows you to use functionality that it already built by other web developers. The use of libraries allows for more efficiency while web developing, and I relied on them heavily in my Coding Session noted down below.
