October 2021

Khan Academy Intro to JS: Drawing & Animation

This module teaches students the basics of JavaScript and introduces them to the Processing JS package, which allows them to create drawings and animations. JavaScript is a commonly used language that may be used in conjunction with HTML to create webpages or for libraries such as p5.js. As a result, laying a firm foundation in this language was important, and this module made that feasible. For me, the most important takeaway from this session was learning about variables, functions, logic and if statements, loops, arrays, objects, and object-oriented design, since I now apply these ideas on a regular basis when coding.

Khan Academy Intro to HTML/CSS: Making webpages

This module provides students with an overview to HTML/CSS, allowing them to develop and design their own websites. It gives students with a solid foundation that can be built upon with the aid of online projects and other resources. In my instance, I proceeded to learn more about web pages by watching instructive Khan Academy videos and visiting sites like Github. As the world moves toward a digital future, there is a lot that can be done with site design.

October 2021

Khan Academy Advanced JS: Games & Visualizations

This module looked at some advanced JavaScript for games and visualisations. Create applications with several scenes, 3D graphics, button menus, and games with scores were among the subjects taught. This module offers a basic introduction to game programming and graphics, since it extends on JavaScript's applications. As a result, I may continue studying in the future by investigating sophisticated games and visualisations and building them by delving further into 3D visuals, employing more complex game structures, desktop editors, and debugging tools. For me, the most important aspect of this subject was learning about buttons and transforms. My understanding of buttons enabled me to increase user involvement in a variety of programmes, including several of my webpages.

October 2021

Khan Academy Advanced JS: Natural Simulations

This module, like Games & Visualizations, built on the students' current JavaScript skills. It's an intriguing unit that blends JS, Processing JS, physics, and mathematical principles to show how the disciplines are related. The course focuses on using oscillations, trigonometry, angular movement, forces, vectors, noise, and randomness to imitate natural trends observable in nature. The substantial emphasis on object-oriented programming was the most important component of this module for me since it helped me to become comfortable with the design, which will be essential when employing complicated data in my projects.
